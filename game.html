Vault Swipe – Game Concept

Vault Swipe is an early-stage puzzle game concept where matching pieces earns coins that players can use to build, decorate, and upgrade their own customizable money vault. This page includes my Canva concept art, UI mockups, and a breakdown of the initial gameplay ideas.

Game Title Concept

Early title logo concept. This is a placeholder to establish the general name and direction of the game.

Start Screen

Initial start screen mockup featuring the “Play” button. This screen will eventually include additional navigation options such as Settings, Vault Preview, and Daily Rewards.

Gameplay Board Concept

Early gameplay concept: themed pieces include coins, cash stacks, rubies, money bags, and dollar bills. Matching pieces earns coins which are saved to the player's vault. The goal is to collect as much currency as possible while clearing tiles and achieving objectives.

Vault Customization

Example vault concept. Players earn coins through gameplay and use them to build, upgrade, and decorate their personal vault structure. The vault evolves visually as the player progresses.

Gameplay Loop

  • Match themed pieces on a puzzle grid.
  • Clear combos to earn coins and rare items.
  • Spend currency to upgrade and customize your vault.
  • Unlock new decorations, themes, and vault rooms as you progress.

Tools & Development

Vault Swipe is currently in the concept and prototyping stage. I’ve used the following tools so far:

  • GDevelop – early 2D prototyping
  • Unity – initial experiments with tile movement
  • Canva – concept art & UI mockups
  • Google Drive / Local assets – organizing early ideas

Future Plans

  • Design full gameplay screen with working match logic.
  • Add animations for tile movement and clearing combos.
  • Build the first version of the vault-building system.
  • Create polished UI elements for menus and upgrade screens.
  • Develop a playable web or mobile prototype.
